# Teslo API

1. Clonar proyecto 
2. ``` npm install ```
3. Clonar el archivo ```.env.example``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno. 
5. Inicializar la base de datos. 
``` 
  docker-compose up -d
```
6. Ejecutar: ```npm run start:dev ```
